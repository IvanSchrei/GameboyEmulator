This is a hobby Project of mine where i try to create a Gameboy Emulator in C++ mostly from scratch (at the moment only very basic external libraries were used).

Feel free to look at the code, get inspired by it, get scared by how bad it is and change it for yourself or just use it to play some of your favorite classics.

I do not condone Piracy. If you use my Project to pirate games (no matter how old) you should take full responsibility yourself.

To load a ROM into the "Gameboy" to run it just change the filepath in /memory/memory_bus.cpp to your desired path.

I might add a better way (.txt file or even GUI) for that and other stuff later but am still working on the basics.

Feel free to let me know about bugs or even new features but i might take some time to respond / implement the changes you wish for.

Also there is a memorytest.exe file in the root of the project that is just there to test it. Feel free to use it or remove, whichever you like best.
